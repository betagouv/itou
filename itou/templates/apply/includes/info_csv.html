{% include "includes/icon.html" with icon="info" %} Les exports sont au format CSV: il s’agit d’un format texte lisible dans Excel et LibreOffice, où chaque valeur est séparée par un virgule. Lorsque vous ouvrez un fichier d’export, pour qu’il s’affiche correctement merci de choisir:

  • Type de séparateur : Virgule (Comma en anglais)
  • Jeu de caractères : Unicode (UTF-16)
